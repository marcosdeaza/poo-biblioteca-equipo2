# 📚 Proyecto Biblioteca Inteligente - Equipo 2

¡Hola! Este es el repositorio para el proyecto de **Programación Orientada a Objetos** (2026) en la Universidad Europea de Valencia. 

El reto: Desarrollar un Sistema de Gestión de Biblioteca en Python aplicando POO, con persistencia de datos (JSON/CSV) y menús por consola.

## 👥 El Equipo y Roles
* **Marcos**: Coordinador y Backend complejo (Lógica de sanciones, excepciones, integración y revisión final).
* **Viviana**: Lógica de negocio (Préstamos, devoluciones, reservas y reglas de negocio).
* **Marc**: Modelo y Persistencia (Clases del dominio y sistema de guardado/carga).
* **Manuel**: Interfaz y Pruebas (Menús de consola, flujo de navegación y testing).
